筆記整理:
	01.複業=以你現有能力開發第二收入來源
	02.兼職不等於複業
	03.複業思維=5W2H
	04.專精和磚頭都需要=T型技能(所以要以電機為專精，部落格把所有其他知識都收集起來)
	05.聽到問題都利用5W2H分析一次(不管帳面上是否是你的責任範圍)，並記錄在部落格中
	06.絕對不能放棄現在工作收入前提下創造額外收入
	
	01.可行的複業點子=已經擁有的技能+多餘時間+已有資源->現實[不能在短期獲利的不要做(需要投入大量資金不要碰)]
	02.複業點子分析指標[高、中、低]: (挑出最優)
		可行性
		說服力
		獲利潛力
		效率
		動力
	03.都謀善斷=[5W2H]+執行力

	目前以存股為最主要
